¿Cómo aprender bien dos idiomas?

Herramientas para fortalecer tanto el kichwa como el español sin dejar uno atrás.

Cambio de Código (Code-switching)

El niño alterna entre dos idiomas en una conversación. Se enseña cada idioma como sistema separado, con la meta de desarrollar competencias diferenciadas.

Ejemplo: “Ñañagu vamos mercadoman”.

Translanguaging

El estudiante usa todos sus recursos lingüísticos como un solo sistema flexible. El enfoque no separa idiomas, sino que mejora ambos como un solo repertorio comunicativo.

Ejemplo: Un niño usa sin esfuerzo kichwa y español según lo que quiere expresar, sin preocuparse por cuál idioma está usando.

Lengua Franca

Idioma que se usa para facilitar la comunicación entre personas con diferentes lenguas maternas. En Otavalo, el español cumple esta función en espacios institucionales.

Ejemplo: En una reunión comunitaria entre kichwahablantes y personas de otras regiones del Ecuador, el español es la lengua común.

Lengua Materna

Idioma aprendido desde la infancia en el entorno familiar. Es la base del pensamiento y la identidad.

Ejemplo: Para muchas familias otavaleñas, el kichwa es la lengua materna que transmite amor, respeto y sabiduría ancestral.

¿Cómo se diseñó esta clase bilingüe?

Esta planificación está pensada para demostrar que sí es posible aprender bien kichwa y español, con estrategias que valoran ambas lenguas y hacen que los estudiantes participen activamente. Se usaron varias mejores prácticas comprobadas:

  • ✔ Partir del contexto real del estudiante: Se trabajó con temas cotidianos del entorno rural como los animales, para conectar lo aprendido con su propia vida.
  • ✔ Uso del translanguaging: El kichwa y el español se usan juntos para construir sentido y fortalecer la identidad sin barreras.
  • ✔ Enseñanza gradual: El docente guía paso a paso hasta que el estudiante pueda expresarse con autonomía.
  • ✔ Actividades multisensoriales: Se usan imágenes, movimiento, sonidos y escritura para apoyar diferentes formas de aprender.
  • ✔ Aprendizaje activo y colaborativo: La clase se construye entre todos: hablando, escribiendo, jugando, reflexionando.

Esta planificación demuestra que enseñar kichwa y español juntos no es un obstáculo, ¡es una oportunidad para aprender mejor!
